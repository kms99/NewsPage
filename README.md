# NewsPage
News api를 이용하여 만든 뉴스 페이지 입니다.

## 프로젝트 소개
JAVASCRIPT강의(codingnoona)에 나온 뉴스페이지 프로젝트를 공부한 코드입니다.

## 주요 항목
pagination, sidebar, 반응형 웹 디자인 (bootstrap), CDN (boopstrap, fontawesome)

## 사용한 API 정보
News API = https://newsapi.org/
(로컬 전용 api로 도메인에서는 적용이 안됩니다.)
